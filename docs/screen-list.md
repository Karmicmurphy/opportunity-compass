# Screen List

## 1. Landing Page

Purpose: Explain what Opportunity Compass is and why it exists.

Core message:

- Find small AI gigs worth touching.
- Avoid the bullshit.
- We translate confusing opportunities into plain-English cards.

Primary action: Start intake.

## 2. Intake Form

Purpose: Learn what kind of opportunities fit the user.

Fields:

- Skill level: beginner, intermediate, advanced
- Available time: 15 minutes, 30 minutes, 1 hour, 2+ hours
- Income goal: $10 today, $25 today, $50 this week, recurring
- Skills: ChatGPT, writing/editing, AI image prompts, AI music prompts, simple web apps, spreadsheets, Canva/design, automation, file cleanup, customer service, review replies, marketplace listings, resume/profile writing

## 3. Opportunity Feed

Purpose: Show opportunity cards with quick verdicts.

Card preview fields:

- Title
- Category
- Beginner Fit
- Verdict
- Estimated Pay
- Estimated Time
- Tools Needed
- Save button

## 4. Opportunity Detail Page

Purpose: Explain one opportunity clearly enough for a beginner to decide whether to act.

Sections:

- What this really means
- Hidden buyer pain
- Why beginners miss it
- Smallest paid offer
- Suggested first message
- Questions to ask
- Deliverable checklist
- Red flags
- Green flags
- Original link placeholder

## 5. Salvage Picks Page

Purpose: Highlight overlooked, niche, lower-competition opportunities.

Each Salvage Pick should explain:

- Why this niche is overlooked
- What buyers need help with
- How to package the service
- What to charge
- Tools to use
- Where to find more like it
- Whether it is beginner-safe

## 6. Daily Top 5 Page

Purpose: Show five curated opportunities or niches for the day.

Each item includes:

- Title
- Verdict
- Why it made the list
- Beginner fit
- Suggested next move

## 7. Saved Opportunities Page

Purpose: Let users keep opportunities they may want to revisit.

Features:

- Saved card list
- Filter by verdict
- Filter by category
- Notes placeholder

## 8. Admin Import Page

Purpose: Let an admin paste raw material to create a draft card.

Inputs:

- Source type
- Source name
- Original link placeholder
- Raw pasted text
- Notes

Action:

- Generate draft opportunity card

## 9. Admin Review Page

Purpose: Let admin edit, approve, reject, or classify draft cards.

Actions:

- Approve
- Edit
- Reject
- Mark Trash
- Mark Scam Risk
- Mark Too Advanced
- Publish to Feed
- Publish to Salvage Picks

## 10. Pricing Page

Purpose: Show free vs paid-style value without fake income promises.

Free:

- Limited opportunity feed
- Basic beginner fit
- Basic scam score

Paid:

- Salvage Picks
- Daily Top 5
- Saved searches
- Suggested replies
- Deliverable templates
- Niche opportunity breakdowns

Suggested pricing:

- $5/month
- $29/year
