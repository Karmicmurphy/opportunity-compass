# Prototype Build Prompt - Opportunity Compass v0.1

Build Opportunity Compass v0.1.

Opportunity Compass is a plain-English opportunity translator and bullshit filter for small AI-enabled work, beginner-friendly freelance tasks, hidden niche gigs, and small-business support packages.

Core promise:

Find small AI gigs worth touching. Avoid the bullshit.

Build a mobile-first static-data prototype using the repo docs and seed data.

Required pages:

1. Landing page
2. Intake form
3. Opportunity feed
4. Opportunity detail page
5. Salvage Picks page
6. Daily Top 5 page
7. Saved cards page
8. Pricing mockup page
9. Admin import mock page
10. Admin review mock page

Use these data files:

- data/opportunities.json
- data/salvage-picks.json
- data/categories.json
- data/risk-labels.json
- data/daily-top-five.json

Opportunity card detail must show:

- title
- category
- verdict
- beginner fit
- estimated pay range
- estimated time
- tools needed
- skills needed
- hidden buyer pain
- what this really means
- why beginners miss it
- smallest paid offer
- suggested first message
- questions to ask
- deliverable checklist
- red flags
- green flags
- risk labels
- boundary note
- where to find buyers
- what not to promise

Also show FlashRiver-ready workbench preview fields when present:

- free tool stack
- local tool option
- cloud tool option
- paid upgrade options
- privacy boundary
- license note
- security note
- module name
- setup steps
- export format
- human review step

Important boundary:

Do not wire real FlashRiver into v0.1.

Do not mention Twis Holo Workshop in the public app.

Do not add scraping, auto-apply, auto-message, real payment, real auth, private imports, or fake income promises.

Use local state for saved cards.

Admin pages can be mock-only.

Design should be plain, practical, blunt, mobile-first, and trustworthy.

The prototype is done when a user can browse, filter, open, save, and understand the cards, and when the mock admin flow clearly shows how opportunities would be reviewed before publishing.
