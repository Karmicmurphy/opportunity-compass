# Opportunity Compass

**Find small AI gigs worth touching. Avoid the bullshit.**

Opportunity Compass is a plain-English opportunity translator and bullshit filter for beginner-friendly AI side gigs, freelance tasks, prompt work, automation micro-gigs, and hidden niche work.

It is **not** a generic job board.
It is **not** a fake side-hustle guru app.
It is **not** a promise of easy money.

The goal is simple:

> Pull scattered AI-related opportunities into one place, explain what they really mean, judge whether they are worth touching, and show beginners the next practical move.

## Core idea

Most platforms show listings. Opportunity Compass translates them.

Each opportunity should answer:

- What is this really asking for?
- Can a beginner realistically do it?
- What tools would they need?
- How long might it take?
- What could they charge?
- What should they say first?
- What red flags matter?
- Is this worth touching, watching, trashing, or avoiding?

## Verdict system

- **Touch** — worth trying.
- **Watch** — maybe useful, but needs caution or more info.
- **Trash** — likely a waste of time.
- **Scam Risk** — possible fraud, exploitation, or unsafe terms.
- **Too Advanced** — real work, but not beginner-safe.

## Public repo safety

This repo is public. Keep it clean.

Do not commit:

- API keys
- tokens
- passwords
- real `.env` files
- private emails
- legal documents
- bank/payment information
- personal addresses or phone numbers
- private screenshots
- real user/customer data
- copied paid-platform content

Use fake/sample data only.
